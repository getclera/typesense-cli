# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-03-18

### Added
- Initial release
- **Collections**: list, get, delete, stats, create, patch, add-field, drop-field
- **Documents**: search, get, delete, delete-by-query, count, export
- **Keys**: list, create, delete, generate-scoped
- **Aliases**: list, get, upsert, delete
- **Health**: health, metrics, debug
- **Sync**: status, verify, facets, drop
- Interactive confirmation prompts for destructive operations
- JSON output support (`--json` flag)
- Environment variable configuration with `.env` file support
